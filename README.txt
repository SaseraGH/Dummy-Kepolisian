
Polisi Case Web (Dummy)
-----------------------
Cara menjalankan:
1. Ekstrak folder police_case_app (atau buka file /mnt/data/police_case_app/index.html langsung di browser).
2. Login menggunakan email yang berakhiran "@email.com" (contoh: user@email.com) dan password apapun.
3. Setelah login akan muncul pop-up selamat datang. Klik Oke.
4. Gunakan tombol "Simulate New Case" untuk menambah kasus baru (akan muncul badge +1 di bel).
5. Klik ikon bel untuk membuka kasus terbaru (diinstruksikan sebagai case paling bawah).
6. Daftar kasus ada di sidebar kiri; klik case paling bawah (border dashed) untuk langsung menuju detail.

File struktur:
- index.html
- styles.css
- app.js
- assets/kepolisian.png (dummy logo, ganti sesuai kebutuhan)
